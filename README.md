# UnderwriteAI 🏦

A RAG-based Insurance Underwriting Assistant powered by LLMs and LangChain.

## Features
- 📄 PDF Policy Upload & Processing
- 📊 Risk Assessment with Score
- ✅ Approval Recommendation with Confidence Score
- 🔍 Missing Information Detection
- 📝 Underwriting Report Generation
- 💬 RAG-based Policy Q&A

## Tech Stack
- Python, Streamlit
- LangChain, RAG Pipeline
- LLM (Gemini/OpenAI)
- FAISS Vector Store
- PyPDF2

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Use Case
Automates insurance policy underwriting by extracting risk insights from documents — similar to enterprise underwriting platforms used in InsurTech.
